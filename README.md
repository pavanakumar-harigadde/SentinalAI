# SentinalAI

## Problem Statement
Detect unauthorized access, insider threats, and anomalous activities from enterprise data access logs.

## Features
- User behavior profiling
- Unauthorized access detection
- Isolation Forest anomaly detection
- Risk scoring engine
- Explainable alerts
- Streamlit dashboard
- Incident report generation

## Technologies
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run

pip install -r requirements.txt

streamlit run app.py
